# 🃏 Blackjack Game (Python)

A command-line Blackjack (21) card game built in Python as part of my Python learning journey.
This project focuses on applying object-oriented programming (OOP) concepts and basic game logic.

---

## 📌 Features

- Playable Blackjack game in the terminal
- Player vs Dealer gameplay
- Object-Oriented Design (OOP)
- Deck shuffling and card dealing
- Dealer card hiding logic
- Blackjack and bust detection
- Ace value handling (11 → 1 when needed)
- Multiple rounds in a single session

---

## 🧠 Concepts Used

- Classes and Objects
- Encapsulation
- Control flow
- Lists and Dictionaries
- User input handling
- Random module
- Game state management

## ▶️ How to Run

### Prerequisites
- Python 3.x installed

### Steps

1. Clone the repository  
   git clone [https://github.com/your-username/blackjack-python.git](https://github.com/amrish407/Blackjack-Game-Python-/tree/main)

2. Navigate to the project folder  
   cd blackjack-python

3. Run the game  
   python blackJackGame.py

---

## 🎮 How to Play

1. Enter the number of games you want to play.
2. You are dealt two cards.
3. Dealer shows one card.
4. Choose Hit (H) or Stand (S).
5. Try to reach 21 without busting.
6. Dealer plays automatically after you stand.

---

## 🧩 Game Rules

- Number cards → face value
- J, Q, K → 10
- Ace → 11 (or 1 if needed)
- Dealer hits until 17+
- Blackjack = 21
